
# Relevance to FAU M.Sc. Artificial Intelligence

## Purpose

This document explains how the KineticAI-AWGC project supports my transition from experimental materials science into artificial intelligence and materials informatics.

## Connection to Artificial Intelligence

The project converts experimentally derived materials data into structured machine-learning datasets.

The current workflow focuses on:

- Data cleaning
- Feature engineering
- Exploratory data analysis
- Phase-fraction prediction
- Bioactivity trend interpretation

These tasks are directly connected to applied machine learning, pattern recognition, and scientific AI.

## Connection to Pattern Recognition

The experimental data used in this project originate from XRD-based phase quantification and SBF bioactivity testing.

The project treats phase evolution as a pattern-recognition problem:
